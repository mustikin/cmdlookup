# CMD Lookup

A searchable cheat sheet of everyday IT and Development commands.

## How to use

1. Open in any browser.
2. Search for commands using the search bar.
3. Filter by category (Network, AD, Disk, User, System, Git).
4. Click the copy icon to copy a command to your clipboard.
5. Add your own custom snippets—they are saved to your browser's local storage.

## Deployment to GitHub Pages

1. Create a new repository on GitHub.
2. Upload the `file index.html` file.
3. Go to **Settings &gt; Pages**.
4. Select `main` branch and `/ (root)` folder, then click **Save**.
5. Your site will be live at `https://<your-username>.github.io/<repo-name>/`.

## Features

- **Responsive Design:** Works on desktop and mobile.
- **Local Storage:** Custom commands persist across sessions.
- **Modern UI:** Built with React, Tailwind CSS, and Lucide icons.
- **Zero Configuration:** No build step required; just open the HTML file.
